# SIMPLE CRUD API

## Tools

* @types/node
  >  npm install @types/node

* nodemon
  > npm install nodemon

* dotenv
  > npm install dotenv

* UUID
  > npm install --save-dev @types/uuid
  > npm install uuid

## How to run

Without nodemon:
  > npm run start

With nodemon:
  > npm run start:dev

Production version:
  > npm run start:prod
